雲端庫存管理系統 (Cloud Inventory)

TypeScript Node.js PostgreSQL Docker AWS

專案背景

許多中小型企業在管理倉儲時仍依賴 Excel 或紙本記錄，這導致了數據滯後與人為錯誤。本專案開發了一個雲端化的 SaaS 平台，讓企業能隨時隨地掌握資產流向。

核心開發重點

  • 資料一致性： 使用資料庫事務 (Transactions) 確保在高併發環境下庫存扣減準確無誤。
  • 自動化預警： 整合 SendGrid API，當品項低於安全庫存時，系統會自動發送補貨通知。
  • 容器化部署： 使用 Docker 封裝應用環境，並部署於 AWS ECS 實現自動擴展。

技術挑戰與學習

在開發初期，大批量資料匯入會導致 API 響應過慢。我後來改用 Redis 緩存機制 處理頻繁讀取的數據，並優化了 SQL 查詢索引，將查詢效率提升了約 60%。
