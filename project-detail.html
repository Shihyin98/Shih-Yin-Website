專案概述

這是一個為中小型企業設計的 SaaS 平台，旨在解決傳統人工清點庫存效率低下且易出錯的問題。系統支援即時庫存追蹤、自動補貨提醒以及多倉儲管理。

核心功能

即時儀表板

透過視覺化圖表掌握當前庫存水位。

自動化通知

低於預設值時自動發送 Email 通知。

API 整合

支援與 Shopify 等電商平台同步。

技術挑戰與解決方案

在開發過程中，處理「併發下單導致的庫存超賣」是最大的挑戰。我透過資料庫的樂觀鎖 (Optimistic Locking) 機制解決了這個問題，確保在高流量下資料的一致性。
